api_flow.py 
from diagflow import Diagram, Node, Edge

d = Diagram()

with d.step("System Overview"):
    api  = Node("API Gateway").fade_in()
    auth = Node("Auth Service").fade_in(at=0.3)
    db   = Node("PostgreSQL").fade_in(at=0.6)
    Edge(api, auth).fade_in(at=0.9)
    Edge(api, db).fade_in(at=1.1)

with d.step("Request Flow"):
    api.send("GET /users", db)

d.export("presentation.html")
Why Diagflow

Everything for visual storytelling

From simple architecture diagrams to complex animated flows — all declarative, all in Python.

⚡

Declarative DSL

Describe diagrams in pure Python. No XML, no YAML, no drag-and-drop. Just clean, readable code that version controls perfectly.

🎬

Step-based Slides

Break your story into steps. Navigate with arrow keys. Each step reveals the next layer of your system, keeping the audience focused.

🔄

Graph Morphing

Smoothly transition between two diagram states. Nodes persist, appear, or dissolve with intelligent semantic matching.

🎨

Beautiful by Default

Glassmorphism, Material, Gradient, and Minimal themes built-in. No configuration needed to produce great-looking output.

📦

Standalone HTML

One file, zero dependencies. Share a single HTML that runs in any browser — no server, no build step, no internet required.

🧩

Auto Layout

Topological, spring, and grid layouts powered by NetworkX. Nodes position themselves so you can focus on the story.

Get Started

Up and running in minutes

$ pip install diagflow
1

Install from PyPI: pip install diagflow — or clone the repo and run pip install -e . for local development.

2

Import Diagram, Node, and Edge from diagflow. Create nodes, connect them, and build your steps.

3

Call d.export("output.html") and open the file in any browser. No server needed — share the single HTML file anywhere.
